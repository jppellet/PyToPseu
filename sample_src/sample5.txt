------------------------|------------------------
x = 0                   |   dans `x`, stocke 0
for i in range(10):     |   répète 10 fois (en comptant avec `i` depuis 0):
    if i % 2 == 0:      |   |   si `i` est un nombre pair:
        x += 2          |   |   |   ajoute 2 à `x`
    if i % 3 == 0:      |   |   si `i` est un multiple de 3:
        x -= 3          |   |   |   diminue `x` de 3
print(x)                |   affiche `x`
